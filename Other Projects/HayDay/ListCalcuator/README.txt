Settings:
1. Choose to alphabetically sort your list by setting 'Sort list:' to either 'True' of 'False'.
2. You can change any number of items by using the 'settings.txt' file. Use the following format: <old value> -> <new value>.
   Example: 8 -> 6. The example will change the number of eight items to six (8 goat milk --> 6 goat milk).

How to use:
1. Put your list in 'input.txt' and run 'ListCalculator.py'. Input must be entered accordingly: <number of items> <item>.
   Example input: 10 brown sugar.
2. Output will be provided in the console and in a text file called: 'output.txt'.
